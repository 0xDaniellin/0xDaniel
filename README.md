# 0xDaniel

Personal website of Daniel —  a DeFi & Web3 content creator and analyst.

 ## About

 This is a static single-page site built with plain HTML & CSS. It features a dark crypto-themed design and displays live Bitcoin & Ethereum prices via the CoinGecko API.

 ## Contents

 - **About Me** — brief intro and focus areas
 - **What I Cover** — DeFi protocol reviews, tokenomics analysis, Web3 trends, educational content
 - **Live Prices** — real-time BTC/ETH prices fetched from CoinGecko
 - **Get in Touch** — link to X (Twitter)

 ## Tech Stack

 - HTML5 / CSS3
 - Vanilla JavaScript
 - CoinGecko public API
